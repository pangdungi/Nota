Section 1

This is the first section. HTML stands for HyperText Markup Language.

Here are some important tags:

  • <h1> to <h6>: Header tags
  • <p>: Paragraph tag
  • <a>: Anchor (link) tag
  • <ul> and <ol>: Unordered and ordered list tags

Section 2

This is the second section. HTML is used to create web pages.

Here are some more tags:

  1. <div>: Division tag
  2. <span>: Span tag
  3. <table>: Table tag
  4. <form>: Form tag

Section 3

This is the third section. Practice makes perfect!

Here are some special characters:

  • &amp;: &
  • &lt;: <
  • &gt;: >
  • &quot;: "
